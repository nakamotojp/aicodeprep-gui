First Launch

After installation, you can run the app two ways:

  • From any terminal: Type aicp and press Enter
  • With a specific folder: Type aicp /path/to/your/project

Quick Start Tutorial

Let's get you making your first AI context in under 2 minutes:

  1. 1 Open your project: Navigate to any coding project folder and type aicp
  2. 2 Review the selection: The app automatically selects relevant files (*.py, *.js, README.md, etc.)
  3. 3 Add a question: Click any preset button or type in the prompt box
  4. 4 Generate: Click the big "GENERATE CONTEXT!" button
  5. 5 Paste: Your code + question is now copied to clipboard - paste it into any AI chat!

File Selection & Management

Smart Defaults

The app starts with intelligent file selection:

‚úÖ Included:
  • Code files (.py, .js, .ts, .html, etc.)
  • Documentation (README.md, CHANGELOG.md)
  • Config files (package.json, pyproject.toml)
‚ùå Excluded:
  • Dependencies (node_modules/, venv/)
  • Build artifacts (dist/, build/)
  • Binary files (images, executables)

Manual Selection

  • Individual Files: Check/uncheck any file
  • Folders: Checking a folder includes all files inside
  • Select All: Includes all non-excluded files
  • Deselect All: Clears all selections
üí° Token Counter: The app includes a token counter to help you stay within AI context limits!

  2. IDE Agent (Step 2): Copy the solution ‚Üí paste into Cursor/Windsurf

    • Use cheaper models for implementation

This gives you the best of both: smart problem-solving + efficient implementation!

‚ùì Help & Troubleshooting

Common Issues

üö´ "Command not found: aicp"

Solution: You need to open a fresh terminal after installation

  • Close all terminal/command windows
  • Open a new one
  • The PATH changes take effect in new sessions only

üì¶ "No module named 'aicodeprep_gui'"

Solution: pipx installation issue

pipx reinstall aicodeprep-gui

üí• App won't start / crashes immediately

Solutions:

  • Update to latest version: pipx upgrade aicodeprep-gui
  • Try debug mode: aicp --debug
  • Delete settings: aicp --delset (removes all saved preferences)

üñ±Ô∏è Right-click menu not appearing (Windows)

Solutions:

  • Run installer as Administrator
  • Restart Windows Explorer: Ctrl+Shift+Esc ‚Üí Find "Windows Explorer" ‚Üí Restart
  • Try the manual registry option in the installer dialog

üìÑ Binary files showing as selected

Solution: This shouldn't happen, but if it does:

  • Click "Deselect All"
  • Click "Select All" - binary files will stay unselected
  • Or manually uncheck them

FAQ

‚ùì Do I need to install this in every project?

A: No! Install once globally with pipx, then use aicp in any folder.

‚ùì What's the difference between aicp and aicodeprep-gui?

A: Same program, aicp is just shorter to type.

‚ùì Can I use this with proprietary/private code?

A: Yes! Everything runs locally, nothing is sent anywhere except when you paste into an AI.

‚ùì How do I update to the newest version?

A: The app checks for updates automatically, or run: pipx upgrade aicodeprep-gui

‚ùì Can I exclude specific files permanently?

A: Create an aicodeprep-gui.toml config file (see Configuration section), or just uncheck them - the app remembers per-project.
